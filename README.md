> **数据可视化课程大作业** · 基于多源数据融合的温州旅游行业数据可视化分析

## 📋 项目简介

本作品以温州旅游行业为研究对象，通过多源数据采集与融合、数据预处理与特征工程、交互式数据可视化等技术手段，构建了一个全面、直观、可交互的温州旅游数据可视化分析平台。

平台包含 **9个功能页面**、**23个交互式图表**、**14条智能推荐路线**，并集成了**实时天气数据**和**智能行程规划**功能。

**在线体验：** [https://wenzhou-travel-viz.streamlit.app](https://wenzhou-travel-viz-g78ygbtvnsqgl9pdvyzty9.streamlit.app/)

---

## ✨ 核心功能

| 页面 | 功能 |
|------|------|
| 📊 **数据总览** | 全局指标展示 + 实时天气速览 + 景点地图 |
| 🌤️ **今日出行指南** | 实时天气 + 景点拥挤度排行 + 高性价比酒店推荐 |
| 🤖 **智能行程规划** | 根据预算/天数/兴趣偏好自动生成最优旅行方案 |
| 🏞️ **景点探索** | 18个景点详情(传说故事+景观介绍+内部游览路线) |
| 🏨 **酒店性价比** | 45家酒店性价比排行 + 价格分布 + 区域对比 |
| 🔗 **游住联动推荐** | 景点→酒店联动推荐 + 区域综合性价比排行 |
| 🗺️ **路线规划** | 14条推荐路线(含天气联动) + 实时拥挤度查询 |
| 📅 **最佳旅行时间** | 月度气候分析 + 四季推荐 |
| 🍜 **美食推荐** | 25家温州特色美食评分排行 + 价格分析 |

### 特色亮点

- 🔄 **实时数据融合**：通过 wttr.in API 获取温州实时天气，动态计算景点拥挤度和酒店价格
- 🧠 **智能行程规划**：基于兴趣匹配、季节适配、区域优化的多目标算法
- 📍 **用户定位**：自动获取位置，计算距景点距离和交通方式
- 📖 **传说故事**：每个景点配有3个民间传说和详细景观介绍
- 🗺️ **内部游览路线**：18个景点均有详细的景区内步行路线指引
- 🌙 **深蓝未来感UI**：渐变背景 + 霓虹光效 + 自然动画过渡

## 🚀 本地运行

```bash
# 1. 克隆仓库
git clone https://github.com/PXT-GH/wenzhou-travel-viz.git
cd wenzhou-travel-viz

# 2. 安装依赖
pip install -r requirements.txt

# 3. 运行应用
streamlit run app.py
```

浏览器自动打开 `http://localhost:8501`

## 📊 数据说明

| 数据类别 | 数据量 | 说明 |
|---------|--------|------|
| 🏞️ 景点数据 | 18个 | 覆盖六大类型(自然风光/历史人文/海岛/乐园/沙滩/古村) |
| 🏨 酒店数据 | 45家 | 覆盖11个区县，3-5星级 |
| 🍜 美食数据 | 25家 | 四大菜系(小吃/瓯菜/海鲜/农家菜) |
| 🚗 交通数据 | 153条 | 景点间距离与驾车时长 |
| 🌤️ 气候数据 | 12个月 | 月均温度、降雨天数、旅游指数 |

**实时数据：** 通过 wttr.in API 获取温州当日实时天气
